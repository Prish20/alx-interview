# 0x06. Star Wars API

## Description

This project interacts with the Star Wars API to retrieve and display characters from a specific movie based on the movie ID passed as a command-line argument.

## Requirements

- Node.js 10.14.x
- Semistandard for linting
- Request module for making HTTP requests
- Command-line arguments for fetching the movie ID

## How to Use

1. Install dependencies:

   ```bash
   npm install request --global
   ```
